# ゲームバランス調整レポート

## 実施日
2026年6月12日

## 目的
敵と味方のバランスを改善し、ゲームの難易度を適切に調整

---

## 1. 敵の強さ調整

### 1.1 基本パラメータ（config.js）

#### 敵の射撃システム
- **射撃間隔**: 180フレーム → **150フレーム**（約20%短縮）
  - 敵がより頻繁に攻撃してくるようになり、プレイヤーへのプレッシャーが増加
  - バトルテンポが上がり、より緊張感のあるゲームプレイに

- **ダメージ**: 18 → **20**（約11%増加）
  - 敵砲の1発当たりのダメージを微増
  - 被弾時の緊張感向上

#### 敵タイプ別HP調整
全敵タイプのHPを段階的に下方修正（インフレ緩和）：
- NORMAL型: 6.0 → **5.0** (20%削減)
- HEAVY型: 12.0 → **10.0** (17%削減)
- SCOUT型: 5.0 → **4.5** (10%削減)
- MAGICAL型: 7.0 → **6.0** (14%削減)
- DEFENSE型: 15.0 → **12.0** (20%削減)
- BOSS型: 18.0 → **16.0** (11%削減)
- TRUE_BOSS型: 25.0 → **22.0** (12%削減)
- SHAKKIN型: 10.0 → **9.0** (10%削減)
- SLIME_KING型: 30.0 → **28.0** (7%削減)

**調整理由**：
- 敵が硬すぎると感じられていた問題を緩和
- 戦闘時間を適切な長さに調整（30〜45秒程度が目標）
- 射撃頻度と攻撃力の微増で難易度は保持

---

### 1.2 侵入者（Invader）の調整（invader.js）

#### 基本敵の変更
- HP: 500 → **450** (10%削減)
- 侵入中の戦闘を約25秒程度の適切な長さに

#### TRUE_BOSS型（真ボス）
- HP: 1800 → **1600** (11%削減)
- 速度: 2.4 → **2.2** (速度微調整)
- 攻撃ダメージ: 22 → **20**
- **調整理由**: 真ボスの難易度が高すぎたため、適切なレベルに引き下げ

#### SPEED型（高速型敵）
- HP: 300 → **280** (7%削減)
- 速度: 3.2 → **3.4** (速度をさらに強化)
- 攻撃ダメージ: （10 → 9に統一予定）
- **調整理由**: HPを下げる代わりに速度で差別化、スピード型らしさを強調

#### POWER型（力強い敵）
- HP: 800 → **750** (6%削減)
- 速度: 1.2 → **1.1** (より重い印象)
- 攻撃ダメージ: 18 → **16**
- **調整理由**: パワータイプは「遅いが強い」というアイデンティティを強化

#### NINJA型（忍者型敵）
- HP: 400 → **350** (12%削減)
- 速度: 2.8 → **3.0** (さらに高速化)
- 攻撃ダメージ: 14 → **12**
- **調整理由**: ニンジャは「敏捷性が特徴」を強調

## 2. 味方の強さ調整

### 2.1 レア度別ステータス調整（config.js：ALLY_RARITY_STATS）

敵が弱くなったため、味方キャラのHP基礎値を微調整して相対的なバランスを取ります：
- ★1: baseHp 100 → **110** (+10%)
- ★2: baseHp 150 → **160** (+7%)
- ★3: baseHp 250 → **270** (+8%)
- ★4: baseHp 400 → **440** (+10%)
- ★5: baseHp 650 → **710** (+9%)
- ★6: baseHp 1200 → **1300** (+8%)
- ★7: baseHp 2500 → **2700** (+8%)
- ★8（スライム王）: baseHp 6000 → **6600** (+10%)

**調整理由**：
- 敵が硬さ削減された分、味方も調整が必要
- 戦闘が短すぎるのを防ぐため、HP増加で戦闘時間を最適化
- 高レア度キャラの優越感を強化

### 2.2 特殊ユニット HP倍率調整（ally.js）

大型ボス級ユニットのHP倍率を微調整：
- **通常大型ユニット**: hpMult 2.5 → **2.8** (+12%)
  - 対象: titan_golem, platinum_golem, dragon_lord など
- **ゴッドキング**: hpMult 5.0 → **5.5** (+10%)
  - 対象: god_king
- **スライム王**: hpMult 10.0 → **11.0** (+10%)
  - 対象: slime_king_god（最強キャラ）

## 3. 難易度別調整

イージーモード：既に十分な難易度緩和（敵ダメージ0.5倍、弾1.5倍）
ノーマルモード：基準値として機能
ハードモード：敵ダメージ1.3倍と弾が減少（0.85倍）で十分なチャレンジ

---

## 4. 期待される効果

### プレイアビリティの向上
- ゲームバランスがより直感的になる
- 敵の多様性が際立つ（速い敵、強い敵、堅い敵など）
- 戦闘時間が適切化（ストレスなく遊べる長さに）

### ゲームの難易度曲線
- 前半ステージ：気持ちよくクリアできる
- 中盤ステージ：適度なチャレンジ感
- 後半ステージ：高度な戦術が必要だがクリア不可能ではない

### 味方キャラクターの役割感
- 敵が弱くなった分、味方を活かしたクリアが実感できる
- 高レア度キャラの価値感が向上

---

## 5. 今後の調整ポイント

### ユーザーフィードバック後に検討すべき項目
1. **プレイ時間**：戦闘が短すぎる/長すぎる場合は射撃間隔を調整
2. **被弾ダメージ**：プレイヤーの緊張感バランス
3. **ステージ進行度合い**：後半ステージの難易度が適正か検証
4. **スペシャル（必殺技）**：ユーザーが使用しやすくなったか確認

## まとめ

このバランス調整は、敵が硬すぎる問題を改善しながら、敵の多様性を活かした戦術的なゲームプレイを実現します。

**調整の哲学**：
- 敵の「タイプ別特性」を活かす
- プレイヤースキルが重要になる難易度
- クリア可能性とチャレンジ性のバランス
