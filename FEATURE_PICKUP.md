# 🚀 追加機能PickUP リスト

## ★★★ 最優先おすすめ機能

### 1. 🏆 実績・トロフィーシステム
- **内容**: 「100体撃破」「天崩地裂を10回発動」などの実績
- **メリット**: 長期的なやりこみ要素・プレイヤーの達成感
- **難易度**: 中
- **ファイル**: `game.js` に `achievements[]` を追加

### 2. 🎯 コンボシステム
- **内容**: 連続命中でコンボボーナス（×1.2 → ×1.5 → ×2.0）
- **メリット**: 技術的なプレイを評価、リプレイ性向上
- **難易度**: 低〜中
- **ファイル**: `battle.js` or `game.js`

### 3. 📱 オンライン通信対戦（リアルタイム）
- **内容**: WebSocket を使った2人対戦
- **メリット**: ゲームの最大の楽しさ
- **難易度**: 高（WebSocketサーバーが必要）

### 4. 🛡️ 装備・アイテムシステム
- **内容**: 「攻撃力+10の兜」「回避率+5%の鎧」などの装備
- **メリット**: 育成の幅が大幅拡大
- **難易度**: 中
- **ファイル**: `config.js` + `ui.js`

## ★★ おすすめ機能（中優先）

### 5. 🗺️ ワールドマップ（ステージ選択改善）
- **内容**: ドット絵風マップ上をキャラが移動してステージ選択
- **メリット**: 世界観の没入感アップ
- **難易度**: 中〜高

### 6. 🔮 フォーメーション（陣形）機能
- **内容**: 「3列陣形」「V字突撃」など仲間の配置パターンを選択
- **メリット**: 戦略性が増す
- **難易度**: 中

### 7. 📊 レーティング・ランクシステム
- **内容**: バトルの評価（S/A/B/Cランク）+ 累積ポイント
- **メリット**: 上手さへのモチベーション
- **難易度**: 低

### 8. 🌟 覚醒（アセンション）システム
- **内容**: 限界突破後に「覚醒」して外見・必殺技が変化
- **例**: タイタンゴーレム → タイタンゴーレム覚醒（翼が生える）
- **難易度**: 中〜高

### 9. ⚡ クイックバトル（AUTO）モード
- **内容**: 仲間がフル自動で戦う「APバトル」
- **メリット**: 周回プレイが楽になる
- **難易度**: 低〜中（`ally.js` を改造）

### 10. 🎪 期間限定イベント
- **内容**: 「クリスマス緊急依頼」「夏祭りバトル」など期間限定ステージ
- **メリット**: 継続的なプレイ促進
- **難易度**: 中
